# Deimageprint.com – Interior Design Portfolio Website

A sleek, modern website built for an interior design company to showcase its services and projects. Designed with an emphasis on visual appeal and mobile responsiveness.

## Features
- Home page with service highlights
- Project gallery with high-quality visuals
- Clean, user-friendly layout
- Contact form for client inquiries
- Optimized for mobile and desktop devices

## Tech Stack
- HTML5
- CSS3
- JavaScript

## How to Use
1. Navigate through the homepage to view key services.
2. Explore the gallery to see completed projects.
3. Use the contact form to reach out for consultations.

✅ Live and accessible at [deimageprint.com](http://deimageprint.com)
